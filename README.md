# Quoc Khanh Ho Portfolio Website

This repository contains my personal portfolio website created for **CSUMB CST 336: Internet Programming**. The website introduces who I am, my interests, and provides basic navigation between the Home, About, and Contact pages.

## Project Overview

This project is a static portfolio website built with HTML and CSS. It demonstrates basic web development skills such as page structure, navigation, image usage, styling, and layout design.

## Pages Included

1. `index.html`
   The main home page of the portfolio website.

2. `about.html`
   A page that shares more information about my background and interests.

3. `contact.html`
   A contact page for visitors to reach out.

4. `css/styles.css`
   The stylesheet used to control the design and layout of the website.

5. `img/`
   Folder containing images used in the website.

## Technologies Used

1. HTML5
2. CSS3
3. Git
4. GitHub Pages

## Features

1. Multi page portfolio layout
2. Navigation menu
3. Personal introduction section
4. About page with interests
5. Contact page
6. Custom CSS styling
7. Images and logo integration

## Website Purpose

The purpose of this website is to practice fundamental web development concepts and create a simple professional portfolio page. This project is part of my coursework for CST 336 at California State University, Monterey Bay.
